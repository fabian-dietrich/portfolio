# fabiandietri.ch

Personal portfolio website for Fabian Dietrich — Interface Designer & Full-Stack Developer.

## Stack

- Vanilla HTML, CSS, JavaScript
- Vite (build tooling)
- Self-hosted fonts: Fraunces, Source Sans 3, DM Mono

## Development
```bash
bun install
bun run dev
```

## Deployment

Built and deployed via Coolify on a Hetzner VPS with Traefik reverse proxy and automatic SSL via Let's Encrypt.
